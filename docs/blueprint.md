# **App Name**: CopyCalc

## Core Features:

- Service Pricing Data: Store service pricelist data in Firestore.
- Real-time Order Calculation: Automatically calculate and update the total order amount.
- Order Management: Save finalized orders with timestamp to the Firestore 'porudzbine' collection.
- Price Lookup: Determine pricing by looking up costs in the Firestore database for printing, scanning, finishing and design.
- Rapid Parameter Input: Use streamlined modular inputs (dropdowns/toggles) to define parameters, trigger Firestore search and automatically find price for Print parameters (Format, Color, Sides).
- Live Order Basket: Display live basket for order with clear descriptions, real time updates, including quick add/remove options.
- Smart Category Navigation: Enable one-click category filtering buttons/icons for main service groups: Print, Scan, Finishing, Design.

## Style Guidelines:

- Primary color: Strong cyan (#0093E9) to convey trust.
- Background color: Very light cyan (#E5F5FE) to promote readability and focus.
- Accent color: Violet (#D000F1) to highlight CTAs, indicating the speed and creativity of the business.
- Body and headline font: 'Inter', a sans-serif font, will provide a modern machined and objective look that is suitable for the body.
- Design the layout to be highly responsive with a clear focus on quick task completion, including quick-access buttons, fixed order panel and rapid reset workflow.
- Develop clear iconography to represent service categories and actions that enables an efficient at-a-glance experience.
- Incorporate micro-interactions on adding and removing items in the basket with smooth transitions to show price changes in real-time.
